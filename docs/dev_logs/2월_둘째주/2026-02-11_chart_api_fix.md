## 1. 주요 변경 사항

### A. KIS 분봉 API 파라미터 수정 (`backend/app/services/market_data.py`)
- **문제**: KIS 분봉 API 호출 시 `fid_input_hour_1` 파라미터가 빈 문자열 → 데이터 0건 반환
- **수정**:
  - `fid_input_hour_1`을 현재 시간(`HHMMSS`)으로 설정
  - `fid_pw_resn_code` → `fid_pw_data_incu_yn` 파라미터 변경
- **결과**: 분봉(1분/5분/10분/30분/60분) 데이터 정상 반환

### B. KIS 분봉 필드 매핑 수정 (`backend/app/services/market_data.py`)
- **문제**: 분봉 응답의 종가/거래량 필드명이 일봉과 다름
  - 분봉: `stck_prpr`(현재가), `cntg_vol`(체결량)
  - 일봉: `stck_clpr`(종가), `acml_vol`(누적거래량)
- **수정**: 분봉/일봉에 따라 동적으로 필드 키 선택
- **결과**: 분봉 close=0.0, volume=0.0 문제 해결

### C. 장 마감 후 일봉 fallback (`backend/app/routers/market.py`)
- **문제**: 장 마감 후 분봉 데이터가 비어있으면 mock 데이터로 fallback → 잘못된 가격 표시
- **수정**: 분봉 데이터가 비어있고 타임프레임이 D/W/M이 아닌 경우, 일봉 데이터로 자동 fallback
- **응답에 `"fallback": "daily"` 표시**하여 프론트엔드에서 구분 가능
