### 구현 내용

#### 1) 분/시간봉 (1분, 5분, 1시간)
- **형식**: `HH:MM` (예: `17:30`, `18:00`)
- Unix timestamp를 한국 시간으로 변환하여 표시

#### 2) 1년봉
- **형식**: `YYYY` (예: `2024`, `2025`, `2026`)
- 연도만 표시

#### 3) 일/주/월봉 (토스증권 스타일)
- **연도 변경 시**: `━ 2026 ━` (시각적 강조)
- **월 변경 시**: `2월`, `12월`
- **일반**: `10일`, `14일`
- 클로저를 사용해 이전 날짜를 추적하여 스마트하게 표시

### 기술적 구현
```typescript
tickMarkFormatter: (() => {
    let prevYear: number | null = null;
    let prevMonth: number | null = null;
    
    return (time: any) => {
        // 분/시간봉: HH:MM
        // 1년봉: YYYY
        // 일/주/월봉: 연도/월/일 스마트 표시
    };
})()
```
