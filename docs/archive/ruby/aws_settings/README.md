# AWS Settings Log

이 폴더는 AWS 관련 확정 설정과 운영 절차를 날짜별로 기록한다.

작성 원칙:
- 파일명은 `YYYY-MM-DD_주제.md`
- 각 문서에는 확정값, 선택 이유, 영향 범위를 함께 기록한다.
- 실제 콘솔/CI 반영 여부를 상태로 남긴다.

권장 섹션:
1. 확정 설정
2. 변경된 리소스명/네이밍 규칙
3. 실패 이력/원인/조치
4. 다음 액션
